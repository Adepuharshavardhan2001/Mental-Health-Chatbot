# CalmMind - Mental Health Support Chatbot

**Live Demo**: https://sentiment---with---cot-from-customer-reviews-kx8vbxhrpcgtybrdv.streamlit.app/  


## Overview
**CalmMind** is a compassionate AI-powered Mental Health Support Assistant built using Google Gemini Generative AI. It provides emotional support, stress management guidance, mindfulness techniques, and positive coping strategies through natural, multi-turn conversations.

## Project Highlights
This project demonstrates **production-grade GenAI chatbot development** with:
- Multi-turn conversational memory
- Domain-specific (Mental Health) prompt engineering
- Empathetic and responsible AI responses
- Clean, modern Streamlit chat interface

## Features
- **Empathetic Responses** — Understands user emotions and responds with care
- **Structured Support** — Covers Emotional Support, Stress Management, Mindfulness & Coping Strategies
- **Suggested Questions** — Helpful prompts to guide users
- **Multi-turn Conversation** — Maintains context throughout the chat
- **Safe & Responsible AI** — Encourages professional help when needed

## Tech Stack
- **Backend**: Google Gemini API + LangChain
- **Frontend**: Streamlit (Chat Interface)
- **Key Techniques**: Advanced System Prompt Engineering, Role-based Instructions, Conversation History Management
